在 IDEA 中添加 Service 窗口，并开启多服务

方式一：

直接使用快捷键 Alt + 8

方式二：

首先选中在上方工具栏中的 View

然后在 Tool Windows 中选中 Services 即可

成功后下方会出现

开启多服务只需要点击最右边的加号图标，然后选择想要添加的服务类型即可

成功后，右键点击想要运行的服务，然后选择 Run 即可
