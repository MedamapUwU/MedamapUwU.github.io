|阅读量:

如何查看 MySQL 中数据的存储情况（我存放的是默认路径即 C 盘）

进入 C 盘，然后在上方工具栏的 查看选项那里，点击并勾选上 隐藏项目按钮

然后文件夹就会出现 Program Data 文件夹

打开后选择 MySQL 文件夹，并打开

点开后会出现两个文件

选择第二个并打开，里面有 Data 文件夹

打开 Data 文件夹后选中你要查看的库，进入即可
