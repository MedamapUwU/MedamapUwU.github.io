Linux 下安装 RabbitMQ Web插件失败的情况

由于 RabbitMQ 是没有客户端的，因此需要在 Linux 系统上安装插件才能使用图形化界面，但是有可能会出现如下报错

主要原因是因为在设置CentOS设置网络和主机名时修改了主机名，而默认的主机名为localhost.localdomain

因此我们需要添加一个 IP 映射的主机名

首先通过 hostnamectl 命令查看当前的 hostname

然后查看一下 etc 目录下的 hosts 文件

发现没有 IP 地址映射到我们的主机名，因此我们需要添加一个 IP 映射
