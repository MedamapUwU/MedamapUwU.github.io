JMeter 安装
发表于2022-08-12
JMeter 安装一、先检查 Java 版本，要求 Java8 以上，在 cmd 中输入 java -version 即可查看 二、从官网下载，选择 zip 文件下载到本地，建议路径为全英文 三、解压到你想要的位置 四、打开环境变量进行配置，可以直接在搜索框中搜索 五、新建两个系统变量变量名： JMETER_HOME 变量值： D:\JMeter\apache-jmeter-5.5 (填上你的安装路径) 变量名：CLASSPATH 变量值：%JMETER_HOME%\lib\ext\ApacheJMeter_core.jar; %JMETER_HOME%\lib\jorphan.jar;（可直接复制，效果如下，如不一致，微调即可） 六、编辑 PATH 变量 添加路径信息 %JMETER_HOME%\bin 七、最后在 cmd 处输入 jmeter 检验是否配置成功 或者打开 bin 目录下的 jmeter.bat 运行 八、大功告成！
JMeter 调整为中文
发表于2022-08-12
JMeter 调整为中文 选择菜单栏的 Options 选项中的 Choose Language 中的 Chinese(Simplified) 修改默认语言，打开 bin 目录下的 jmeter.properties 文件 在 40 行的位置（我这里是第40行，有可能不一样）添加上 language = zh_CN 保存即可
